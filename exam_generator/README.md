# 考试试卷生成器

这是一个用于从Excel表格读取试题信息，并按照指定顺序生成Word试卷的桌面应用程序。

## 功能特点

- 从Excel文件中读取试题数据
- 支持按题号区间选择试题
- 可选择是否打乱选项顺序
- 自动生成包含题目、选项、分值、答案和解析的完整试卷
- 提供直观的图形用户界面

## 技术栈

- Python 3.x
- pandas - 用于读取Excel文件
- python-docx - 用于生成Word文档
- tkinter - 用于创建图形用户界面
- openpyxl - 用于Excel文件的读写支持

## 项目结构

```
exam_generator/
├── main.py              # 主程序文件，包含GUI界面和程序入口
├── question.py          # 试题类模块，定义试题数据结构
├── excel_reader.py      # Excel读取模块，负责从Excel中读取试题
├── word_generator.py    # Word生成模块，负责生成Word试卷
├── requirements.txt     # 项目依赖列表
└── README.md            # 项目说明文档
```

## 安装说明

1. 确保已安装Python 3.6或更高版本

2. 安装所需依赖：

   ```bash
   pip install -r requirements.txt
   ```
   或者
   ```bash
   pip install pandas openpyxl python-docx
   ```

## 使用方法

1. 准备Excel试题文件，确保包含以下列：
   - 题号
   - 题目
   - 题型
   - 选项A
   - 选项B
   - 选项C
   - 选项D
   - 分值
   - 正确选项
   - 解析
   - 阅读理解的文章（可选）
   - 备注（可选）

2. 运行程序：

   ```bash
   python main.py
   ```

3. 在图形界面中：
   - 点击"浏览..."按钮选择Excel试题文件
   - 点击"获取题号范围"按钮自动填充题号范围
   - 可手动修改起始题号和结束题号
   - 勾选"选项乱序"复选框可随机打乱选项顺序
   - 点击"生成试卷"按钮开始生成Word试卷
   - 选择保存路径后，程序将生成Word试卷文件

## Excel文件格式要求

请确保Excel文件具有以下格式：

| 题号 | 题目 | 题型 | 选项A | 选项B | 选项C | 选项D | 分值 | 正确选项 | 解析 | 阅读理解的文章 | 备注 |
|------|------|------|-------|-------|-------|-------|------|----------|------|----------------|------|
| 1 | 示例题目1 | 单选题 | 选项A内容 | 选项B内容 | 选项C内容 | 选项D内容 | 2.0 | A | 本题解析... | (可选) | (可选) |
| 2 | 示例题目2 | 多选题 | 选项A内容 | 选项B内容 | 选项C内容 | 选项D内容 | 3.0 | AB | 本题解析... | (可选) | (可选) |

## 注意事项

- 确保Excel文件编码正确，避免出现乱码
- 题号应为数字格式，便于程序排序和筛选
- 选项内容为空时，程序会自动跳过该选项
- 生成的Word文档包含三个部分：题目、参考答案和试题解析

## 故障排除

- 如果程序无法读取Excel文件，请检查文件路径和格式是否正确
- 如果生成的Word文档格式不正确，请检查Excel中的数据格式
- 如果遇到依赖问题，请确保所有必要的Python包已正确安装
